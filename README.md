# 项目简介

## 核心功能
- 用户注册与登录
- 财务预算管理
- 收入信息管理
- 支出信息管理
- 财务统计
- 系统首页管理
- 公告信息管理
- 部门管理

## 适用场景
- 企业财务管理
- 财务人员日常操作
- 管理员权限管理

## 技术特点
- 架构：B/S、MVC、Web框架
- 开发环境：IDEA、JDK1.8、Mysql、Maven、Navcat
- 技术栈：Java、SpringBoot、Vue
